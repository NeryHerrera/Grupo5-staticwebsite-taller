# Grupo5-staticwebsite-taller

Aplicación React creada con [Vite](https://vite.dev/) para el taller [Host a Static Website](https://docs.aws.amazon.com/hands-on/latest/host-static-website/host-static-website.html) de AWS.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build para producción

```bash
npm run build
```

Los archivos generados quedan en la carpeta `dist/`.
